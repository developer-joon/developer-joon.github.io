2026년 4월, AI 업계에 굵직한 뉴스들이 쏟아졌습니다. Anthropic의 차세대 모델 유출부터 OpenAI의 역사적 모델 퇴출, Apple의 AI 개방 정책까지 — 기술 업계를 뒤흔든 6가지 이슈를 정리합니다.

## 1. Anthropic Claude Mythos 유출 — Opus 4.6을 넘는 차세대 모델 존재 확인

Anthropic의 내부 문서가 유출되면서, 현재 최고 성능 모델인 Claude Opus 4.6보다 상위 계층인 **Claude Mythos** 프로젝트가 존재한다는 사실이 확인됐습니다.

### 주요 내용
- **Mythos는 Opus 위 티어**: 현재 공개된 Sonnet/Opus 라인업 위에 또 다른 프리미엄 모델이 준비 중
- **2026년 하반기 출시 예정**: 내부 로드맵에 따르면 Q3~Q4 사이 제한된 베타 테스트 시작
- **멀티모달 강화**: 비전, 오디오, 코드 생성 모두 대폭 개선된 통합 모델로 알려짐
- **가격 미정**: Opus 4.6의 2~3배 가격대로 예상됨

이번 유출은 Fortune과 Bloomberg가 확인했으며, Anthropic은 공식 성명을 내지 않았지만 내부 보안 점검을 강화했다고 전해집니다.

## 2. OpenAI GPT-4o 완전 퇴출 (4/3) — 2년 만에 퇴역, GPT-5.4 mini/nano 대체

OpenAI는 4월 3일, **GPT-4o 모델을 완전히 퇴출**한다고 공식 발표했습니다. 2024년 출시 이후 약 2년간 서비스됐던 GPT-4o는 GPT-5.4 mini와 nano로 대체됩니다.

### 왜 퇴출했나?
- **비용 효율**: GPT-5.4 mini는 GPT-4o보다 추론 비용이 40% 저렴하면서도 성능은 동등 이상
- **API 통합 단순화**: 5.x 시리즈로 라인업 정리 (GPT-3.5/4는 이미 레거시 상태)
- **개발자 피드백**: 실제로 GPT-4o를 쓰는 앱이 급감했고, 대부분 GPT-5.1/5.4로 전환 완료

### 마이그레이션 일정
- **2026년 4월 3일**: GPT-4o API 신규 요청 중단
- **2026년 6월 30일**: 완전 종료 (기존 앱도 마이그레이션 필수)

OpenAI는 기존 GPT-4o 사용자에게 자동 마이그레이션 도구와 GPT-5.4 mini 크레딧을 제공합니다.

## 3. Apple Siri AI 개방 (iOS 27) — Gemini, Claude까지 Siri에서 접근

Apple이 **iOS 27**에서 Siri를 서드파티 AI 모델에 개방한다고 발표했습니다. 이제 iPhone 사용자는 Siri를 통해 Google Gemini, Anthropic Claude, OpenAI GPT 등 원하는 AI 어시스턴트를 선택해 사용할 수 있습니다.

### 주요 기능
- **AI 선택 옵션**: 설정 > Siri > AI 제공자 선택 메뉴 추가
- **통합 음성 인터페이스**: "Hey Siri, ask Claude..." 형태로 호출 가능
- **프라이버시 유지**: 온디바이스 처리는 Apple Neural Engine, 클라우드 요청은 사용자 동의 후 전송

이는 Apple이 자체 AI 모델만 고집하지 않고, **사용자 선택권을 최우선**으로 한다는 전략 전환으로 해석됩니다. 9to5Mac은 "Apple의 AI 오픈 플랫폼 선언"이라고 평가했습니다.

## 4. NVIDIA Vera Rubin — Groq 3 LPU 통합, 토큰 비용 1/10

NVIDIA가 새로운 AI 추론 가속 플랫폼 **Vera Rubin**을 공개했습니다. 이번 플랫폼은 Groq 3 LPU(Language Processing Unit)를 통합해 기존 GPU 대비 **토큰당 비용을 1/10 수준**으로 낮췄습니다.

### 주요 스펙
- **Groq 3 LPU 통합**: Groq의 전용 언어 처리 칩을 NVIDIA 인프라에 네이티브 탑재
- **추론 속도 10배 향상**: GPT-5 클래스 모델을 실시간 스트리밍 수준으로 처리
- **토큰 비용 90% 절감**: 클라우드 AI 서비스 제공업체의 운영 비용 대폭 감소
- **2026년 Q3 출시**: 베타 프로그램은 5월부터 시작

NVIDIA는 "AI 추론이 이제 진짜 대중화된다"며, 소규모 스타트업도 대규모 AI 서비스를 저렴하게 운영할 수 있게 됐다고 강조했습니다.

## 5. Meta Avocado 비공개 전환 — 오픈소스 아이콘의 폐쇄형 선회

Meta가 차세대 AI 모델 **Avocado**를 **비공개로 전환**한다고 발표했습니다. Llama 시리즈로 오픈소스 AI를 선도했던 Meta의 전략 변화가 주목받고 있습니다.

### 왜 비공개로?
- **수익화 압박**: 오픈소스 Llama는 인기를 끌었지만, 직접적인 수익은 거의 없었음
- **경쟁 심화**: OpenAI, Anthropic 등 폐쇄형 모델들이 기술 격차를 벌리자 대응 전략 수정
- **내부 기술 유출 방지**: Llama 기반 파인튜닝 모델들이 Meta의 경쟁자로 성장한 점도 고려

### Avocado 특징
- **멀티모달 + 에이전트 기능**: 비전, 오디오, 코드, 액션 실행까지 통합
- **API 전용 제공**: 가중치 공개 없이 API 형태로만 서비스
- **2026년 6월 출시 예정**: Meta AI 플랫폼에 먼저 탑재 후 점진적 개방

오픈소스 커뮤니티는 Meta의 이번 결정에 실망감을 표했지만, 일각에서는 "지속 가능한 AI 개발을 위한 불가피한 선택"이라는 평가도 나옵니다.

## 6. OpenAI Sora API 종료 — AI 영상 생성의 현실

OpenAI가 **Sora API를 종료**한다고 발표했습니다. 2024년 공개 당시 큰 화제를 모았던 AI 영상 생성 모델이지만, 상용화에는 실패한 것으로 평가됩니다.

### 종료 이유
- **낮은 사용률**: 예상보다 실제 API 호출량이 적었음 (월 평균 10만 건 미만)
- **높은 운영 비용**: 영상 생성은 텍스트 생성 대비 컴퓨팅 비용이 100배 이상
- **품질 이슈**: 실제 프로덕션 수준 영상 생성에는 여전히 부족하다는 피드백 다수

### 대안은?
- **RunwayML Gen-3**: 현재 가장 인기 있는 AI 영상 생성 서비스
- **Pika Labs**: 실시간 편집 기능이 강점인 스타트업
- **Stability AI Video**: 오픈소스 기반으로 커뮤니티 지원 활발

OpenAI는 "Sora는 연구 프로젝트로서의 역할을 다했다"며, 향후 GPT 시리즈에 집중할 것이라고 밝혔습니다.

## 마무리: AI 산업의 재편성

2026년 4월, AI 업계는 **기술 격차 확대**와 **수익화 압박** 사이에서 빠르게 재편되고 있습니다.

- **Anthropic, NVIDIA**: 기술 혁신으로 격차 벌리기
- **OpenAI**: 레거시 정리와 효율화에 집중
- **Apple**: 개방형 플랫폼으로 사용자 선택권 강화
- **Meta**: 오픈소스에서 폐쇄형으로 전략 선회

AI는 더 이상 "무엇이든 할 수 있다"는 기대가 아닌, **실질적 가치를 증명해야 하는 단계**에 접어들었습니다. 앞으로 몇 달간 이 트렌드가 어떻게 전개될지 주목됩니다.
